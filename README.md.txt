# Les tests d'hypothèse avec python

	- Tester une proportion avec une valeur hypothétique
	- Tester la difference de moyennes entre deux groupes d'échantillon avec t-test
	- Le faire avec plus de deux groupes(ANOVA) et réaliser les pairwise t-test
	- Tester la difference de PROPORTIONS entre deux groupes d'échantillon avec les test de proportion
	- Utilisation du chi carré pour les tests d'indépendance ou d'ajustement 
	- Contraintes des tests paramétriques
	- Les tests non paramétriques   

les librairies utilisées sont:

	- pingouin
	- scipy.stats
